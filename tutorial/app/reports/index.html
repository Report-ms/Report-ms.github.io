Отчёты

Секция Отчёты определяет интерактивные отчёты для пользователей.

В секции Отчёты содержится список из элементов, заголовок которых определяет название отчёта, например Аналитика заказов, Заявки и т.д.

Элемент Отчёт​

НазваниеТипОписание
Доступно для ролейМассив строкОтчёт будет доступен пользователям с указанными ролями.
ИконкаИконкаЭта иконка будет отображаться в меню пользователя.
СтраницыМассив объектов Страница отчётаСтраницы отчёта будут отображаться в отдельных вкладках.

Элемент Страница отчёта​

Название страницы будет соответствовать названию вкладки.

НазваниеТипОписание
ЭлементыМассив объектов Элемент страницы отчётаКакие интерактивные элементы находятся на странице.

Элемент страницы отчёта​

Элемент страницы отчёта будет отображаться на странице как интерактивный элемент.

Название элемента может быть одним из значений:

Значение​

Статистическое значение по выбранному полю для данных справочника, например количество пользователей, сумма заказов и т.д.

Круговая диаграмма​

Круговая диаграмма по выбранному полю для данных справочника в разрезе поля для заголовка, например количество пользователей по ролям, сумма заказов по категории и т.д.

Гистограмма​

Столбчатая диаграмма по выбранному полю для данных справочника, где каждый столбец соответствует полю для заголовка, например количество пользователей по ролям, сумма заказов по категориям и т.д.

Воронка​

Отображает последовательность этапов в процессе и количество объектов (например, лидов, заказов, клиентов), которые перешли на каждый этап или выбыли из процесса. Позволяет визуализировать и анализировать эффективность конверсии на каждом шаге.

Пример: Воронка продаж по этапам: "Входящая заявка" (100 шт.) -> "Квалификация" (80 шт.) -> "Презентация" (50 шт.) -> "Закрытая сделка" (30 шт.).

Фильтр​

Интерактивный элемент управления, который позволяет пользователю динамически фильтровать данные на всей странице отчёта или в отдельных её элементах.

Карта​

Визуализирует географическое распределение данных. Отображает метки, тепловые карты или регионы на основе географических привязок.

Диаграмма Ганта​

Используется для планирования и управления проектами. Отображает задачи в виде горизонтальных полос вдоль временной шкалы, показывая их длительность, последовательность и overlap.

Индикатор​

Отображает ключевой показатель (KPI) в виде одного числа, часто сопровождаемого значком тренда (рост/падение) по сравнению с предыдущим периодом или плановым значением.

Фигуры​

Позволяет размещать на странице отчёта статические декоративные элементы или простые фигуры (линии, прямоугольники, стрелки) для визуального разделения или акцента на других элементах, улучшения компоновки и восприятия страницы.

Веб элемент​

Позволяет встраивать на страницу отчёта сторонний веб-контент с помощью iframe. Например, можно embed видеоролик, карту из другого сервиса, форму опроса или дашборд из другой системы.

Кнопка действия​

Интерактивная кнопка, которая выполняет заданное действие при нажатии. Используется для автоматизации рутинных операций без перехода в карточку объекта.

Название параметра задаёт под каким идентификатором он будет извлечен из запроса.
