Report.ms - A platform for rapid development of enterprise applications

What is it Report.ms ?

Report.ms — it is a powerful and flexible platform designed for the development of enterprise applications that automate the business processes of organizations.

It allows programmers to quickly create and modify applications using the easy-to-understand YAML format and integration with Python.

What for?

  • Process automation: Accelerate the development and implementation of business applications to automate everyday tasks.
  • Flexibility: Easily change the structure of the application and immediately check the result.
  • Intuitive interface: Generation of modern web interfaces with access control and support for static pages.
  • BI-reports: Create interactive reports for data visualization.

For whom?

  • Programmers: Those with basic knowledge of Python who want to quickly develop applications and automate tasks.
  • Project Managers: Those in need of flexible tools for managing business processes.
  • Analysts: Who seek to gain access to interactive data visualization.

Main functions

  • Templates and custom RMS files for describing the structure of applications.
  • Python support for implementing business logic.
  • A convenient extension for VS Code for code editing and application integrity control.
  • A Docker container for configuring and running an application in a productive environment.
  • Easy debugging of applications.
